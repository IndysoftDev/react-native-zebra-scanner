# react-native-zebra-scanner

Zebra SDK wrapper for Android and iOS

## Installation

```sh
npm install react-native-zebra-scanner
```

## Usage

```js
import ZebraScanner from "react-native-zebra-scanner";

// ...

const result = await ZebraScanner.multiply(3, 7);
```
